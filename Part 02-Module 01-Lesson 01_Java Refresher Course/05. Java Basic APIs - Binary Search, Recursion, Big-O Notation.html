Big-O Notation Quiz1

What is the time complexity for searching a target in an unsorted 2D m * n matrix
SOLUTION: O(m * n)

Big-O Notation Quiz2

What is the time complexity of the following function fun()?

int fun(int n) 
{     
    for (int i = 1; i <= n; i++) 
    { 
        for (int j = 1; j < n; j += i) 
        { 
            // Some O(1) task 
        } 
    }     
}
SOLUTION: O(nlog(n))
